# HTML Basics Internship Project

This project is created as part of a Web Development internship task.
It demonstrates basic HTML concepts with hands-on practice.

## Concepts Covered
- HTML document structure
- Headings, paragraphs, links, and images
- Forms using input fields and textarea
- HTML attributes (id, href, src)
- Semantic HTML elements
- Simple navigation menu

## Tools Used
- HTML5
- Visual Studio Code
- Web Browser

## Folder Structure
- index.html – Main HTML file
- images/ – Image assets
- requirements.txt – Optional file
